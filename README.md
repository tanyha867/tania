Random Task Generator
Приложение для генерации случайных задач с возможностью фильтрации по типам, добавления новых задач и просмотра истории сгенерированных заданий.
Описание
Random Task Generator — десктоп приложение на Python с графическим интерфейсом (Tkinter), которое:
•	генерирует случайные задачи из предустановленного или пользовательского списка;
•	фильтрует задачи по категориям: «Учёба», «Спорт», «Работа», «Другое»;
•	позволяет добавлять новые задачи прямо в интерфейсе;
•	отображает историю сгенерированных задач;
•	сохраняет список задач в JSON файл (tasks.json) для повторного использования;
•	очищает историю задач по команде пользователя.
Ключевые возможности:
•	интуитивно понятный графический интерфейс;
•	мгновенная фильтрация задач и истории по выбранному типу;
•	сохранение пользовательских задач между запусками программы;
•	визуальное отображение текущей сгенерированной задачи и полной истории;
•	обработка ошибок (например, предупреждение при попытке сгенерировать задачу из пустого списка).
Используемые технологии:
•	Python 3;
•	Tkinter (графический интерфейс);
•	JSON (хранение данных);
•	random (генерация случайных значений).

Примеры использования
Базовый сценарий
1.	Запустите приложение.
2.	Нажмите кнопку «Сгенерировать задачу» — в центральной области появится случайная задача с указанием типа в скобках, например: Прочитать статью (Учёба).
3.	Повторите нажатие, чтобы увидеть другие задачи.
4.	Посмотрите на список в разделе «История задач» — там отображаются все сгенерированные задания в хронологическом порядке.
Фильтрация задач
1.	В выпадающем списке «Фильтр по типу» выберите, например, «Спорт».
2.	Нажмите «Сгенерировать задачу». Теперь приложение будет выбирать задачи только из категории «Спорт» (например, Сделать зарядку (Спорт)).
3.	Выберите «Все» в фильтре, чтобы вернуться к генерации из полного списка.
Добавление новой задачи
1.	Введите текст новой задачи в поле «Новая задача», например: «Подготовить презентацию».
2.	Выберите тип задачи в соседнем выпадающем списке, например, «Работа».
3.	Нажмите кнопку «Добавить задачу». Вы увидите сообщение «Задача добавлена!».
4.	Новая задача появится в общем пуле и может быть сгенерирована в дальнейшем.
Очистка истории
1.	Сгенерируйте несколько задач, чтобы заполнить историю.
2.	Нажмите кнопку «Очистить историю». Список в разделе «История задач» станет пустым.
Проверка сохранения данных
1.	Добавьте 1–2 новые задачи через интерфейс.
2.	Закройте приложение.
3.	Запустите его снова.
4.	Проверьте файл tasks.json в папке с программой — в нём должны быть сохранены все задачи, включая добавленные вами.
5.	Нажмите «Сгенерировать задачу» — среди вариантов будут и ваши новые задачи.
________________________________________
Запуск приложения
1.	Убедитесь, что у вас установлен Python 3.x.
2.	Сохраните код приложения в файл, например, task_generator.py.
3.	Поместите файл в удобную директорию.
4.	Запустите приложение командой:
bash
python task_generator.py
Приложение автоматически создаст файл tasks.json, если его ещё нет, и заполнит его предустановленными задачами.
________________________________________
Примечания
•	Файл tasks.json создаётся в той же директории, где находится скрипт task_generator.py.
•	История задач хранится в памяти приложения и не сохраняется между запусками.
•	Для корректной работы с кириллицей убедитесь, что кодировка файла — UTF 8.
